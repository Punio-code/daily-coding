# Java課題リポジトリ

このリポジトリは、**Java 学習用の課題プロジェクト**です。  
- 課題を解くためのモジュール（`exercises`）
- 解答例のモジュール（`solutions`）

に分かれています。

## モジュール構成

- **exercises**
  - ここに問題が配置されていますので、自身のプロジェクトにファイルをコピー&ペーストして**問題を解いてください**。
- **solutions**
  - **解答の見本**です。**唯一の正解ではありません。**
  - 設計・実装は複数のアプローチがあり得ます。あくまで**正解例の一つ**として参考にしてください。

> 🔎 解答を見る前に、まずは `exercises` で自力で取り組むことをおすすめします。  
> 自身のコードと解答を比較するときは「なぜ違いが出たのか」「自分のコードの良いところ・改善できるところ」を言語化できると学習効果が高まります。
