# Java課題リポジトリ

このリポジトリは、**Java 学習用の課題プロジェクト**です。

- 課題を解くためのモジュール（`exercises`）
- 解答例のモジュール（`solutions`）

に分かれています。

## モジュール構成

- **exercises**
    - ここに問題が配置されていますので、自身のプロジェクトにファイルをコピー&ペーストして**問題を解いてください**。
- **solutions**
    - **解答の見本**です。**唯一の正解ではありません。**
    - 設計・実装は複数のアプローチがあり得ます。あくまで**正解例の一つ**として参考にしてください。

- **practice**
  - 動画講座の内容をまとめております。

> 🔎 解答を見る前に、まずは `exercises` で自力で取り組むことをおすすめします。  
> 自身のコードと解答を比較するときは「なぜ違いが出たのか」「自分のコードの良いところ・改善できるところ」を言語化できると学習効果が高まります。

## Gitコマンド（初学者向け）

このリポジトリでよく使うGitコマンドをまとめています。

### 状態確認

| コマンド         | 説明        |
|--------------|-----------|
| `git status` | 変更ファイルの確認 |
| `git diff`   | 変更内容の確認   |

### 変更の記録

| コマンド                      | 説明          |
|---------------------------|-------------|
| `git add .`               | 変更をまとめてステージ |
| `git commit -m "message"` | 変更内容を記録     |

### 履歴の確認

| コマンド                | 説明           |
|---------------------|--------------|
| `git log --oneline` | コミット履歴を簡潔に表示 |

### リモート操作

| コマンド       | 説明        |
|------------|-----------|
| `git push` | GitHubに反映 |
| `git pull` | 最新状態を取得   |

※ 危険な操作（`reset --hard` 等）は記載していません。
